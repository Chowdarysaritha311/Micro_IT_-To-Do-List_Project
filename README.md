# To-Do List Web App

## Description
A simple and interactive web-based To-Do List to manage daily tasks. Users can add, mark, and delete tasks efficiently.

## Features
- Add new tasks
- Mark tasks as complete/incomplete
- Delete tasks
- Responsive and user-friendly design

## Tech Stack
- HTML
- CSS
- JavaScript
